Lights

Allows you to list and control individual lights. You can only change the state of one light at a time. For controlling multiple lights with one API call, see Groups.

  • List all lights and their names
  • Show light information
  • Changing light color and turning them on/off

List all lights and their names

GET /api/username/lights

Response

{
  "1": {
    "name": "TV Left"
  },
  "2": {
    "name": "TV Right"
  }
}

The number before each light is it’s ID. You’ll use it for changing the state of the light.

Show light information

GET /api/username/lights/number

Parameters

number assigned to the light you wish to control, see List all lights and their names.

Changing light color and turning them on/off

PUT /api/username/lights/number/state

Parameters (courtesy of Hacking the Hue)

on
true if the light should be on.
bri
brightness, in range 0 - 254. 0 is not off.
hue
hue, in range 0 - 65535.
sat
saturation, in range 0 - 254.
xy
color as array of xy-coordinates.
ct
white color temperature, 154 - 500.
alert
select flashes light once, lselect flashes repeatedly for 10 seconds.

Input

{
  "on": false
}

Response

[
  {
    "success": {
      "/lights/1/state/on": false
    }
  }
]
